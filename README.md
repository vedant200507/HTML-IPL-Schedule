# 🏏 HTML IPL Schedule

A mini project built using **only HTML5** to demonstrate the practical implementation of various HTML elements. This project presents the IPL 2026 Match Schedule with team details, player information, stadium details, Google Maps integration, and embedded match highlights.

---

## 📌 Project Overview

This project was developed using only **HTML5** without any CSS or JavaScript. The primary objective was to practice and demonstrate the use of different HTML tags while creating an interactive IPL Match Schedule website.

The project includes multiple IPL matches along with team information, player lists, stadium details, embedded Google Maps, and YouTube highlights.

---

## ✨ Features

- 🏏 IPL 2026 Match Schedule
- 👥 Team and Player Information
- 🏟 Stadium Details
- 🗺 Google Maps Integration
- 🎥 Embedded YouTube Match Highlights
- 📷 Team Images
- 📋 Tables for Organized Data
- 📂 Expandable Sections using `<details>` and `<summary>`
- 🔗 Hyperlinks and Embedded Content
- 📄 Structured HTML5 Layout
